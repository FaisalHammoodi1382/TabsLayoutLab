## TabsLayoutLab — Android Application

This Android Studio project implements a **two-tab interface** using **TabLayout and ViewPager2**.

### Tab 1 — LeftFa

* User selects a color using radio buttons.
* Submit button shows a Toast displaying the selected color.
* Selected color is passed to Tab 2 using Fragment Result API.

### Tab 2 — Rightdi

* Displays received color text dynamically.
* Shows "NO DATA" when no selection exists.
* Includes sports selection using checkboxes.
* SPORT button displays selections using AlertDialog.
* Displays "NO SPORT" when nothing is selected.
