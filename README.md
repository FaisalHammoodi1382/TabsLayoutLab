## TabsLayoutLab (FaisalLab6)

This Android Studio project demonstrates a **TabLayout + ViewPager2** application with **two fragments**:

- **LeftFa (Tab 1):** lets the user pick a color and submit it.
- **Rightdi (Tab 2):** receives the selected color from Tab 1 and shows it on-screen. It also includes an **AlertDialog with multi‑choice checkboxes** (used for the lab screenshots: some selected vs none selected).

### Key features (Lab requirements)
- ViewPager2 adapter with two fragments (Tab 1 / Tab 2)
- Data passed from **LeftFa → Rightdi** using Fragment Result API
- AlertDialog with multi-choice items + OK button
- English + French resources (values + values-fr)
- Target SDK / Min SDK set in Gradle (see `FaisalLab6/build.gradle`)
- Custom launcher icon resources in `res/mipmap-*`
